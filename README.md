# 🏟️ SportNest

SportNest is a modern, dynamic, and user-friendly Sports Facility Booking Platform. It bridges the gap between sports enthusiasts and facility owners, allowing users to discover, explore, and seamlessly book sports turfs, courts, pools, and gyms, while empowering owners to manage their properties effortlessly.

---

## 🎯 Purpose

The primary objective of **SportNest** is to digitize and simplify the sports venue reservation process. 
- **For Users:** Eliminates the hassle of making phone calls or physically visiting venues to check availability. It provides real-time slot viewing and instant booking management.
- **For Owners:** Offers a comprehensive dashboard to list, update, view, and delete sports facilities, ensuring efficient business operations and resource management.

## ✨ Features

### 👤 User Features
- **Interactive Discovery:** Browse a wide variety of sports facilities including Football Turfs, Cricket Nets, Badminton Courts, and Swimming Pools.
- **Advanced Filtering & Search:** Search venues by name or filter them by specific sport types instantly.
- **Real-Time Slot Booking:** View available time slots and book them based on hourly capacity.
- **Personal Booking Dashboard:** View, update (change date/time), or cancel bookings anytime from a personalized panel.

### 🏢 Owner/Admin Features
- **Facility Management:** A robust CRUD (Create, Read, Update, Delete) dashboard to manage registered venues.
- **Dynamic Badge System:** Beautifully categorized badges for different sports types with responsive mobile and desktop table layouts.
- **Secure Authentication:** Next-generation session handling and secure token-based routes.

The client-side is built with modern tools to ensure speed, accessibility, and responsiveness:

| Package Name | Purpose |

### ⚙️ Back-End (Node.js / Express)

| Package Name | Purpose |
| :--- | :--- |
| `express` | Minimalist and flexible web application framework for building APIs. |
